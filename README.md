# FirstAssignment #Task1

## Overview
This is a basic Python script that takes two numerical inputs from the user and performs fundamental arithmetic operations: addition, subtraction, multiplication, and division. It's a simple command-line utility to demonstrate basic input/output and arithmetic in Python.

## Features
*   Takes two numbers as input.
*   Calculates and displays the sum.
*   Calculates and displays the difference.
*   Calculates and displays the product.
*   Calculates and displays the quotient.
